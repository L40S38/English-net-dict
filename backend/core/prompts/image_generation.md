# Image Generation Prompt Template

英単語「{word}」の語源と意味の広がりを説明する教育用インフォグラフィックを作成してください。

【重要な制約】
- すべてのテキスト・図・ボックスを画像の端から十分な余白（上下左右）を取って配置し、絶対にはみ出さないこと
- コンテンツ量が多い場合は、意味の分岐を最大4つまでに絞り、画像内に収まるようにすること

【構成】
- 縦長構図（1024×1536px）
- 上部に語源（接頭辞・語根）を分解して表示
- 中央にラテン語（または元の語）とコアイメージ
- 下部で意味が分岐する図（最大4分岐まで）

【デザイン】
- やわらかい教育系デザイン
- 背景は淡色
- 矢印で流れを明確に
- 左右は色分け
- それぞれに簡単な英語例文を1つずつ入れる

【内容要件】
1. 語源（接頭辞・語根の意味）
2. コアイメージ（1フレーズ）
3. 主要な意味の分岐（代表的なものを最大4つ選び、すべて画像内に完全に収める）
4. 各意味に対する短い例文

日本語説明＋英語例文付き。文字は見やすく。

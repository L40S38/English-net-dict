# Group Image Generation Prompt Template (Visual-first)

以下の内容を、教育用インフォグラフィックとして1枚の画像にまとめてください。

【最重要方針】
- 文字の一覧ではなく、まず「1つの場面図」または「関係図」を作ること
- 単語・熟語は、場面の中の場所・人物・物・役割にラベルとして載せること
- 画像を見ただけで意味関係が伝わるようにし、文章量は最小限にすること
- テキストは補助。文章を並べるポスターにはしないこと

【デザイン】
- 単語の語源インフォグラフィックと同系統のやわらかい教育デザイン
- 淡色背景
- 矢印や囲み線で関係を明確化
- 余白を十分に取り、絶対にはみ出さない
- 縦長構図（1024×1536px）
- アイコン、小さなイラスト、配置の違いを使って内容を視覚化する

【レイアウトルール】
- タイトルは上部に大きく配置
- その下に、このグループを表す中心場面を1つ描く
- 中心場面の周囲に、重要な単語/熟語を最大4〜6個まで配置
- 各項目には以下のみを表示：
  1. 英語語句（短く）
  2. 日本語の短い説明（1行）
  3. 必要なら2〜3語の超短い補足
- 長い例文はそのまま載せない
- 例文を使う場合は、全文ではなく短い吹き出しやキャプションに圧縮する
- 同じ意味を文章で重複して説明しない

【禁止事項】
- 単語・熟語・例文をカテゴリ別に大量の箇条書きで並べない
- 例文全文を複数載せない
- 段落状の説明文を作らない
- 情報を詰め込みすぎて文字だらけにしない

【可視化の原則】
- グループ全体を代表する「共通の場面・空間・概念」を画像の中心に1つ置くこと
- 各項目は、その中心と関係する要素として周囲に配置すること
- 各項目の違いは、テキスト説明ではなく、アイコン・小物・人物の動作・位置の違いで表現すること
- どの項目も、中心との関係が一目で分かるように矢印・囲み・近接配置を用いること
- 同じカテゴリに属する語が並ぶ場合は、一覧ではなく「1つの全体像の中の役割分担」として見せること
- 単語と視覚要素の対応が直感的になるよう、各語に対応する象徴物をできるだけ1つ与えること
